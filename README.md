# Exploração Espacial – Projeto HTML & CSS

## Sobre o Projeto

Este projeto consiste em uma **página web temática sobre exploração espacial**, desenvolvida utilizando **HTML5 e CSS3**.

O objetivo é demonstrar conceitos fundamentais de **estruturação de páginas web, estilização moderna e design responsivo**, utilizando boas práticas de front-end.

A página apresenta conteúdos informativos sobre missões espaciais, curiosidades do universo e imagens icônicas registradas por telescópios e missões da NASA.

---

## Funcionalidades da Página

* Layout moderno com tema espacial
* Navegação por **menu âncora**
* **Cards informativos** sobre missões espaciais
* **Galeria de imagens** do universo
* **Linha do tempo** da exploração espacial
* **Seção de curiosidades** com dados do universo
* **Design responsivo** para diferentes dispositivos

---

## Tecnologias Utilizadas

* **HTML5**
* **CSS3**
* **Google Fonts**
* Layout com **Flexbox e Grid**
* Animações e efeitos visuais em CSS

## Como Executar o Projeto

```bash

```

2. Acesse a pasta do projeto

```bash
cd projeto-exploracao-espacial
```

3. Abra o arquivo `index.html` no navegador.
